Description of Project

I want to use a method developed in humans and mice in butterflies, but I don’t know that the assay will work because the butterfly genome has a different G/C content than humans and mice, and the assay uses a set of restriction enzymes whose recognition sites might be affected by different genome contents.

Analysis Goal

I want to make sure that using the assay’s restriction enzymes will function as expected in the butterfly genome to produce sequenceable sizes - with the majority not over 400bp, but also not under 20bp, since I can’t reliably recover that fragment size. I need to ensure that the fragment pattern is the same between the butterfly and mouse genomes. There is a second version of the protocol that uses slightly different restriction enzymes that I will also investigate.

Conclusions

The density curves from the in silico digest on the butterfly genome show that a larger fraction of the butterfly genome fragments will be under <20bp relative to the mouse genome. It looks like the genome contents of the butterfly genome will change how this assay works no matter what, but fewer <20bp fragments will be lost if I use the first version of the protocol, which does not use the TTAA cutter. There is a much stronger effect between the two protocols than in the mouse genome.

Prioritized follow-up studies

To follow up on this, I will actually perform the experiment on the butterfly samples using the version 1 protocol, since I learned that fewer <20bp fragments will be lost in that protocol (which intuitively makes sense given the enzymes used). I will also evaluate the efficiency of using those restriction enzymes by examining the actual distribution of fragment lengths on a gel relative to what I expected from the in silico digestion to see if there is suboptimal digestion.
