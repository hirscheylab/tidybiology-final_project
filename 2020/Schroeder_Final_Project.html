DATA VISUALIZATION Next it’s time to visualize the number of daily hospitalizations at a National and State (NC) level. Because it is not possible to add a break in our scale using ggplot2, I have included two graphs to enlarge the data for NC.

P1 <- ggplot(data=Combined_DATA, aes(x= date))+

CONCLUSIONS While it appears that the percentage of positive covid cases trends between the National and State level, there is a significant increase in positive rates from about 6% to 15% beginning in April. This is most likely due to the “first covid wave” where the number of tests performed increased drastically as more tests became available.

PRIORITIZED FOLLOW_UP STUDIES While it appears that you can use NC data as a predictor for National covid positive case rates, it does not provide specific questions about individual outcomes.
